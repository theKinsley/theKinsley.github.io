Digital Image Processing

Feb 20, 2023

  • 采样vs量化：采样和量化是相反的
    尖锐区域->细腻采样、粗糙量化
    平滑区域->粗糙采样、细腻采样

  • 增强vs复原：图像增强是主观的、图像复原是客观的（面向传输等过程导致的损坏）

  • 连通性
    条件：(1)位置相邻、(2)灰度值相似
    m连通？连通是一条线

  • 点运算

    1. 反转变换
    2. 对数变换：动态压缩[0, 1.5*10^6]的傅立叶频谱图像素值至[0, 255]
    3. 伽马变换
    4. 对比度拉伸
    5. 灰度级切片：对关注与不关注两个范围的像素值分开处理
    6. 位平面(Bit-plane)切片
      • 高位(7)：大纲、主要内容
      • 低位(0)：细节
      • *与傅立叶图频谱图的高、低频异曲同工

  • 代数运算

    1. 算术运算：+-*/
      • 去除叠加性噪声
    2. 逻辑运算：~|&^

  • 直方图

    • 定义：p(r_k) = n_k / n
      • n是像素总数
      • k是特定像素值
      • r是灰度级(rank)
    • 缺点：缺乏空间关系

  • 重要应用：镜头检测方法等
    1。 直方图均衡化

    • 主旨：使图像的像素均衡占有所有可能灰度级，以具备高对比）
    1. 直方图相减：相邻视频帧处理
    2. 时空切片：相邻灰度级处理，分析颜色、纹理改变，甚至物体运动的改变

  • 空间滤波

    1. 线性滤波
      • 均值(average)滤波器：去除噪声，容易丧失细节
      • 中值(median)滤波器：去除噪声，更好地保留细节，适合消除椒盐噪声
      • 最大值滤波器：突出亮处
      • 最小值滤波器：突出暗处
    2. 非线形滤波
      • 锐化滤波器
        1. 拉普拉斯变换

  • 检索指标

    1. MRR
    2. ANMRR
    3. MAP
    4. P-R Curve
